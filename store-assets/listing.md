• 核心永久免费。训练器、范围图表、错题复习永远免费；Pro 只解锁进阶工具，绝不锁基础。

免费内容
• 翻前训练：开局 / 防守 / 面对 3-bet / 面对 4-bet / 挤压 / 冷跟 / 推弃，现金与锦标赛。
• 13×13 范围图表，逐手研究。
• 智能出题：弃 / 加 / 跟 / 全下 题型均衡的题库，弃牌只取临界牌，不抽废牌。
• 错题复习堆 + 轻量间隔重复，漏洞一直可练。
• 翻前诊断：在真实牌桌上做 18 或 45 手测试，每题给答案 + 讲解，最后出个性化报告。

Pro 解锁
• 个人画像 + 漏洞分析——你的倾向（太松 / 太紧 / 被动 / 激进）、最常踩的坑、一键去练。
• 20 天个性化训练计划，每日可执行卡片 + 连续打卡。
• 我们自算的推弃 Nash（9 人 8–20bb、6 人 10–20bb、BB 跟全下、单挑 5–25bb）——标注实测可利用度，不靠假设。
• 范围 vs 范围胜率计算器，可选填翻牌 / 转牌 / 河牌。

由一只讨厌假 GTO 的松鼠打造。诚实 > 花哨。

说明：这是学习与训练工具，非真钱赌博。

誠実なプリフロップGTO練習＋個人のリーク診断＋20日プラン。オフライン対応。

**詳細な説明（≤4000）：**
プリフロップを「なんとなく」で打つのはもう終わり。プリフロップGTOポーカーは、レンジを直感に変え、あなたの弱点（リーク）を正直に教えます。

ここが違う
• 誠実さが土台。EVや混合頻度を捏造しません。手調整のレンジは「参考レンジ」と明記し、実際に計算できる局面（プッシュ/フォールドのオールインエクイティ）だけ本物のNash数値を表示します。
• オフライン対応。トレーナー全体が端末内で動作——飛行機でも電車でも、どこでも練習できます。
• コアは永久無料。トレーナー・レンジ表・ミス復習は無料のまま。Proは上級ツールだけを解放し、基本機能はロックしません。

無料でできること
• プリフロップ練習：オープン / ディフェンス / vs 3ベット / vs 4ベット / スクイーズ / コールドコール / プッシュフォールド、キャッシュ＆MTT。
• 13×13レンジ表をハンドごとに学習。
• スマート出題：レイズ / コール / フォールド / オールインをバランスよく、フォールドは境界ハンドのみ。
• ミス復習スタック＋軽い間隔反復で、弱点を練習し続けられます。
• プリフロップ診断：実際のテーブルで18または45ハンドのテスト、毎ハンド解答＋解説、最後に個別レポート。

Pro

• 20日間の個別トレーニングプラン、毎日のカード＋連続記録。
